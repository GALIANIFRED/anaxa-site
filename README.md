# Bureau ANAXA — version optimisée

## Lancer en local

```bash
npm install
npm run dev
```

Puis ouvrir http://localhost:3000

## Déployer sur Vercel

1. Créer un compte sur Vercel
2. Importer ce dossier
3. Lancer le déploiement
4. Connecter ensuite le domaine `www.bureauanaxa.com`
